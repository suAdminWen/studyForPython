# 说明

《大话设计模式》示例代码，python的实现。

> python 版本3.6.6

# 设计模式的原则：

- 开放-封闭原则
- 单一职责原则
- 依赖转置原则
- 里氏替换原则
- 迪米特法则
- 接口隔离原则
